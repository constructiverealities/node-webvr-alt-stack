Pass through the HTC Vive camera video to the HMD.

## Usage

Camera footer overlay - so you can see your feet:

    npm run demo-footer

Using THREE.WebGLRenderer({*alpha:true*}),

    npm run demo-alpha

No three.js, just simple stand-alone html:

    npm run demo-raw
